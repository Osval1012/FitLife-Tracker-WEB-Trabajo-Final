# FitLife-Tracker-WEB-Trabajo-Final
1. Propósito del Sitio Web

El sitio web de FitLife Tracker presenta de manera clara y moderna la información principal del proyecto. Su objetivo es mostrar las funciones, diseño y contenido relacionado con la propuesta, utilizando buenas prácticas de HTML y CSS.

2. Estructura del Proyecto

El sitio está compuesto por 5 páginas, todas conectadas mediante un menú de navegación:

index.html – Página principal

acerca.html – Información sobre el proyecto

nutricion.html – Ayuda en Alimentos

blog.html – Tips y Ayuda para empezar en el mundo del ejercicio

contacto.html – Formulario de contacto

Carpetas adicionales:

css/estilos.css – Archivo de estilos

img/ – Carpeta de imágenes (logo e imágenes utilizadas)

3. Tecnologías Utilizadas

HTML5 – Estructura del sitio

CSS – Estilos y diseño visual

JavaScript – Validaciones o interactividad 

Google Fonts – Tipografías Poppins e Inter

5. Cómo Ejecutar el Proyecto

Descarga el repositorio o usa git clone.

Mantén la estructura de carpetas.

Abre index.html en cualquier navegador: Chrome, Firefox, Edge o Safari.

Asegúrate de que la carpeta img esté en el mismo nivel que el resto de los archivos.

No requiere instalación adicional.

6. Navegación del Sitio

El menú superior permite moverse fácilmente entre las páginas:

Inicio – Introducción y presentación general

Acerca – Propósito del proyecto

Nutricion – Ayuda En Alimentacion

Blog – Tips y Ayuda

Contacto – Formulario para mensajes

7. Tecnologias Utilizadas

Se usaron Visual Studio y mi computadora personal

 8. Conclusión

Este proyecto demuestra el uso correcto de HTML, CSS y diseño web básico para crear un sitio organizado, funcional y visualmente atractivo. El sitio web FitLife Tracker responde a la estructura solicitada en el curso y presenta una interfaz clara y fácil de navegar.
